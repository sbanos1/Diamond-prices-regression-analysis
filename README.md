This project analyzes the factors that most strongly influence diamond prices using the 2022 Diamonds dataset. The primary goal is to build an accurate and interpretable regression model that predicts price based on both numerical and categorical characteristics such as carat, clarity, color, and cut. Through exploratory data analysis, hypothesis testing, model diagnostics, transformations, and stepwise regression, the project identifies the most significant predictors while addressing issues such as multicollinearity and non-constant variance. Ultimately, the project develops a best-fit model that explains how key diamond attributes contribute to price variation and provides reliable estimates for future pricing predictions.
